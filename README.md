# CommerceForge Docs

Standalone documentation project for CommerceForge.

## Scripts

```bash
npm install
npm run dev
npm run build
```

## Structure

- `src/pages/index.astro` docs homepage
- `src/pages/guides/*` guide pages
- `src/layouts/DocsLayout.astro` shared docs layout
- `src/lib/docs.ts` sidebar and navigation config
